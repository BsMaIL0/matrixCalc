# 🧮 Матричный калькулятор — консольное приложение на C++

Этот проект реализует вычисление определителя и обратной матрицы для квадратной матрицы, заданной в файле. Программа работает с вещественными числами и не использует STL.

## ⚙️ Функциональность

- **Вычисление определителя** (ключ `-det`)
- **Вычисление обратной матрицы** (ключ `-inv`)
- Поддержка квадратных матриц произвольного размера
- Обработка вырожденных матриц и ошибок ввода

## 🖥️ Использование

```
./matrixcalc -det input.txt output.txt
./matrixcalc -inv input.txt output.txt
```

# 🧮 Матричный калькулятор — консольное приложение на C++

Этот проект реализует вычисление определителя и обратной матрицы для квадратной матрицы, заданной в файле. Программа работает с вещественными числами и не использует STL.

## ⚙️ Функциональность

- **Вычисление определителя** (ключ `-det`)
- **Вычисление обратной матрицы** (ключ `-inv`)
- Поддержка квадратных матриц произвольного размера
- Обработка вырожденных матриц и ошибок ввода

## 🖥️ Использование

```
./matrixcalc -det input.txt output.txt
./matrixcalc -inv input.txt output.txt
```

## 📁 Формат входного/выходного файла

Первая строка: `rows cols` (два целых числа)

Последующие строки: элементы матрицы построчно

**Пример (input.txt):**

```
3 3
2 -1 0
-1 2 -1
0 -1 2
```

## 📤 Выходные данные

- Для `-det`: одно число — определитель, или `error`
- Для `-inv`: обратная матрица, или `singular`/`error`

## 🛠️ Технические детали

- Чистый C++ (без STL: `vector`, `string` и т.д.)
- Вычисление определителя методом Гаусса с выбором главного элемента
- Вычисление обратной матрицы методом Гаусса‑Жордана
- Динамическое выделение памяти под матрицы (`double**`)
- Сравнение строк через самодельную функцию `comp()`

## 📁 Структура проекта

```
├── matrixCalc.h // заголовочный файл с объявлениями
├── matrixCalc.cpp // реализация всех функций
├── main.cpp // точка входа, парсинг аргументов
└── README.md // этот файл
```


## 🚀 Сборка и запуск

Скомпилируй с поддержкой C++11 или новее:

```
g++ -std=c++11 main.cpp matrixCalc.cpp -o matrixcalc
./matrixcalc -det input.txt output.txt
```
